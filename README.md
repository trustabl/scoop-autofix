# scoop-autofix

Scoop bucket for [autofix](https://github.com/trustabl/autofix) — AI-powered behavioral attestation and fix generation for Python agent tools.

## Install

```powershell
scoop bucket add trustabl https://github.com/trustabl/scoop-autofix
scoop install autofix
```

## Usage

```bash
autofix fix --source ./my-tool --model gpt-4o
autofix version
```

For full documentation, see [github.com/trustabl/autofix](https://github.com/trustabl/autofix).
